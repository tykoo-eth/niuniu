# 🐂 斗牛（牛牛）多人在线扑克游戏

## 快速开始

```bash
npm install
npm start
```

打开浏览器访问 http://localhost:3000

多人游戏：在同一局域网内，其他人访问 http://你的IP:3000

## 游戏规则

- 2-6人游戏，每人发5张牌
- 抢庄 → 选倍数 → 发牌 → 分牌 → 比大小
- 将5张牌分为3+2两组，3张之和必须是10的倍数（有牛）
- 剩余2张之和的个位数决定牛的大小

### 牌型排名（从小到大）
没牛 < 牛丁 < 牛二 < ... < 牛九 < 牛牛 < 四炸 < 五花牛 < 五小牛

### 倍数规则
- 牛七及以下：1倍
- 牛八、牛九：2倍
- 牛牛：3倍
- 四炸：4倍
- 五花牛：5倍
- 五小牛：8倍
